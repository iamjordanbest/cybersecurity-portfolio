ğŸ”§ TROUBLESHOOTING
================================================================================

PROBLEM: Port 8501 already in use
SOLUTION: Use a different port
   streamlit run src/dashboard/app.py --server.port 8502
   Then open: http://localhost:8502

PROBLEM: ModuleNotFoundError
SOLUTION: Install missing packages
   pip install streamlit plotly pandas pyyaml

PROBLEM: Database not found
SOLUTION: Verify database exists at:
   data/processed/grc_analytics.db
   If missing, run:
   python scripts/initialize_database.py
   python src/ingestion/run_all_ingestion.py
   python scripts/generate_mock_compliance_data.py

PROBLEM: Dashboard loads but shows errors
SOLUTION: Test the analytics modules
   python scripts/test_analytics.py

PROBLEM: Slow performance
SOLUTION: 
   - Close other browser tabs
   - Clear browser cache
   - Restart the Streamlit server
